install node js from their official documentation

npm comes with node installation (node package manager)

install angular
>>npm install -g @angular/cli

list out all the versions
>>ng --version

creates a new project with default routing set to true and stylesheets are css files.
>>ng new project-name --routing --style=css

To run the development server
>>ng serve
>>ng serve --open (opens the browser with the server link)
